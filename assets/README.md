# Art Drop Folder

Drop optional game art here. The browser will use these filenames automatically when present.

## Scene Art

- `setup-vineyard.jpg` - 1600 x 900 px, opening setup background. Wide vineyard estate image.
- `vineyard.jpg` - 1600 x 700 px, vineyard rows / slopes / field operations.
- `cellar.jpg` - 1600 x 700 px, barrels, tanks, bottling, or cellar work.
- `tasting-room.jpg` - 1600 x 700 px, commercial side: tasting room, restaurant table, distributor tasting, or sales room.

Scene art should be landscape, realistic or painterly-realistic, warm natural light, not too dark, with clear subject matter. Keep important detail away from the lower-left corner because the game overlays short white text there.
