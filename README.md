# Cellar Baron

A no-build browser strategy game about running a wine estate through five volatile years.

Open `index.html` in a browser to play. The game saves to `localStorage` from the in-game Save button.

Optional art can be dropped into `assets/`. See `assets/README.md` for filenames and sizes.

## Core Loop

- Choose a region, grape, and production philosophy.
- Choose a difficulty tier, including `Grand Cru` for a tighter cashflow game.
- Spend monthly action placements on seasonally available vineyard, cellar, bottling, sales, hospitality, finance, or estate construction work.
- Manage winery cashflow through rent, interest, operating burn, credit-line draws, and debt repayment.
- Set bottle price against demand and buyer ceilings.
- Accept and fulfill buyer contracts while reserving inventory for committed orders.
- Hire named staff with asymmetric bonuses and payroll pressure.
- Survive weather, supply, market, critic, and personnel events.

Win by building a benchmark estate before month 60: high prestige, strong net worth, and enough fulfilled contracts.
